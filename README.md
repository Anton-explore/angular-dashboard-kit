## Installation
Clone or download this project on your PC.
Open terminal.
Switch to project directory.
Use `npm install` or `npm i` to install dependencies.

## Usage
To start project run `ng serve` in terminal. 

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.

To run test server use `json-server --watch mocked-data.json` command in directory with `mocked-data.json` file.  
Server will be running on port 3000. (http://localhost:3000)
